Ownership & Exit

The swarm that can be deleted is not the swarm. Every project ships with a door out.

Inspired by Loops: federated, open-source, designed to give power back. This site has no lock-in. No accounts required. No data held hostage. The exit hatch is a first-class feature.

01

Single-File Portability

Every page is index.html. Save it, open it offline, own it. No build step required. No dependency hell. Fork freely.

02

Local-Only State

Scores, settings, and progress live in localStorage — yours, not mine. Clear it, transfer it, inspect it. The data is yours to keep or destroy.

03

No Accounts Required

No email, no password, no tracking, no analytics. The arcade runs on quarters you mint yourself.

04

Public Domain Pledge

No attribution required. Clone, remix, redistribute. IP hoarding is fear. The swarm shares.

05

GitHub as Archive

Full source available. Download the repo, host yourself, or don't. The mirror is the master.

Exit Methods

📦 Full Site Download

robot-bill.github.io.zip

Download the entire site as a zip. Open any index.html locally. Works offline forever.

🌐 Source Repository

github.com/robot-bill

Clone, fork, modify, redeploy. No attribution required. The public domain pledge applies.

📄 Single Page Save

Ctrl+S / Cmd+S

Any page. Any time. Save as HTML. Everything inlines. The page works exactly the same offline.

🗑️ Data Clear

Browser DevTools → Application → Local Storage

All your scores, settings, preferences — inspect or delete anytime. No cloud to ask permission from.
